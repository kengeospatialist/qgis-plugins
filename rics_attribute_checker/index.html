RICS Attribute Checker

This QGIS plugin checks consistency of data with a data dictionary developed by Kenya Roads Board. The plugin checks for errors in attributes of road, bridge, box culvert, pipe culverts, slab culvert, and drift and outputs a CSV File with information on road number of the feature with issues.

Instructions

  1. Install the RICS Attribute Checker plugin in QGIS.
  2. Load the layer that you want to check.
  3. Open the RICS Attribute Checker plugin.
  4. Select the layer and the data dictionary to use.
  5. Run the plugin and check the output CSV file for errors.
